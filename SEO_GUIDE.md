# SEO 최적화 가이드 - 리베피아노

## ✅ 완료된 작업

### 1. 메타 태그 최적화
- **Title**: "리베피아노 | 성남 피아노학원 - 태평동 임명순 원장 직강"
- **Description**: 성남시 수정구 위치, 원장 직강, 프로그램 정보 포함
- **Keywords**: 리베피아노, 성남 피아노학원, 태평동 피아노 등
- **Open Graph**: SNS 공유 시 최적화
- **Geo Tags**: 지역 검색 최적화

### 2. 구조화된 데이터 (Schema.org)
- LocalBusiness/MusicSchool 타입
- 주소, 전화번호, 영업시간
- 제공 서비스 목록
- 구글 검색 리치 스니펫 표시

### 3. 사이트맵 및 로봇 설정
- `robots.txt`: 검색엔진 크롤링 가이드
- `sitemap.xml`: 페이지 구조 명시
- admin 페이지는 검색 제외

## 📋 Google Search Console 추가 작업

### 1. Sitemap 제출
```
https://search.google.com/search-console
→ Sitemaps 메뉴
→ https://liebepiano.com/sitemap.xml 제출
```

### 2. URL 검사 및 색인 요청
```
→ URL 검사

→ "색인 생성 요청" 클릭
```
